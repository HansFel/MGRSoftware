► SO STARTEN SIE:

  1. Doppelklick auf "Maschinengemeinschaft.exe"
  
  2. Im Fenster: Klicken Sie auf den grünen Button
     "➕ Neue Datenbank erstellen"
     Speichern Sie die Datei (z.B. als "Uebung.db")
  
  3. Der Server startet automatisch
  
  4. Ihr Browser öffnet sich auf http://127.0.0.1:5000
  
  5. Melden Sie sich an:
     Benutzername: admin
     Passwort:     admin123


► WAS IST DAS?

  Sie läuft nur auf Ihrem Computer und Sie können alle Funktionen
  testen, ohne die echten Daten zu verändern.
