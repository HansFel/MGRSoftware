Alle nicht benötigten Dateien, Testskripte, Migrationen, temporäre Datenbanken und Build-Ordner wurden entfernt.

## Installation auf dem Server

1. Kopiere das Projekt auf den Server.
2. Führe das Skript `install.sh` aus:
   ```bash
   bash install.sh
   ```
3. Starte die Anwendung:
   ```bash
   source .venv/bin/activate
   python3 web_app.py
   ```

Weitere Hinweise siehe `install.sh`.
